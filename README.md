### Description
----
This is a REST API developed using Node.js that implememts the basic feed post functionalities. Following are the functionalities available:

* SignUp / Login / JWT Auth
* Create/ Read / Update/ Delete Posts


### Requirements
----
* Node.js

### Documentation
----

```

```
